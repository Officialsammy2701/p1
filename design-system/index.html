07 Brand & Logo

Each variation ships in two fills — dark-bg (black paths, for use on dark surfaces) and light-bg (white paths, for use on light surfaces). Left pane = dark, right pane = light.

Master Mark

Primary icon · use standalone at any size · favicons, app icons, embossing

Simplified Mark

Reduced-detail icon · optimised for small sizes, sub-32px usage

Accent Variant

Special use only · Burnt Sienna (#E07A3F) · limited to favicon, special applications, embossing on branded materials

Wordmark

Logotype only · use when the icon isn't needed; headers, footers, email signatures

Full Lockup

Icon + wordmark · primary brand expression; presentations, proposals, hero areas

Usage Guidelines

Dark surfaces

Use the light-path files (e.g. logo_master.svg) — they render with white fills that show against dark backgrounds

Light surfaces

Use the black_variation files (e.g. logo_master_black_variation.svg) — dark fills for light backgrounds

Minimum size

Master mark: 24px minimum. Simplified mark: 16px minimum. Below these sizes, use text-only wordmark

Burnt Sienna accent

Special use only. Reserved for favicon (32px), premium embossing, and featured callouts. Do not use as primary logo.

No modifications

Do not rotate, recolour, stretch, add effects, or place on busy backgrounds without a backing shape

Theme switching

In web contexts, toggle between variants using the .light class on <html> — exactly as ThemeToggle.tsx does

Clear Space Rules

Clear space = "h" (logo height) on all sides. For a 100px logo, maintain 100px breathing room. This ensures the mark stands independently and isn't visually crowded by adjacent content.
